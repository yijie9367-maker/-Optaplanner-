# AI智能排课 — 学生端微信小程序 操作指引手册

> **用途**：本手册用于配置智能体AI，使其能够指引客户正确使用「AI智能排课」微信小程序。
> **适用角色**：学生用户
> **主色调**：`#4A90D9` 蓝色系
> **版本**：V1.0

---

## 一、小程序概述

「AI智能排课」是一款面向大学生的智能排课系统配套小程序，帮助学生便捷地查看课表、考试安排、参与论坛交流、管理个人资料。

### 三大核心模块（底部Tab栏）

| Tab | 图标 | 功能 |
|-----|------|------|
| 📅 课表 | 日历图标 | 查看每周课程安排 |
| 💬 论坛 | 对话气泡图标 | 校园讨论社区 |
| 👤 我的 | 人物头像图标 | 登录/个人资料 |

### 页面结构总览

```
课表（首页Tab） ─── 考试安排（子页面）
论坛（Tab2）    ─── 帖子详情 ─── 发帖编辑器
我的（Tab3）
```

## 二、准备工作

### 2.1 如何进入小程序

1. 打开微信App
2. 点击底部「发现」→「小程序」
3. 搜索小程序名称进入
4. 或扫描小程序二维码直接进入

### 2.2 首次使用引导

- 首次打开小程序，会进入**课表页面**
- 如果未登录，课表页面会显示**空课表 + 登录引导弹窗**
- 弹窗提示"欢迎使用AI智能排课"，可选择「立即登录」或「稍后再说」
- 点击「立即登录」会跳转到「我的」Tab页进行登录

### 2.3 权限说明

| 功能 | 未登录 | 已登录 |
|------|--------|--------|
| 查看课表 | ❌ 显示空课表 | ✅ 正常查看 |
| 查看考试安排 | ✅ 可查看演示数据 | ✅ 查看真实数据 |
| 浏览论坛帖子 | ✅ 正常浏览 | ✅ 正常浏览 |
| 搜索帖子 | ✅ 可用 | ✅ 可用 |
| 点赞帖子 | ❌ 弹窗提示登录 | ✅ 可用 |
| 发表评论 | ❌ 弹窗提示登录 | ✅ 可用 |
| 发布帖子 | ❌ FAB按钮隐藏 | ✅ 可用 |
| 查看个人资料 | ❌ 显示登录表单 | ✅ 显示资料 |

## 三、登录指南

### 3.1 如何登录

1. 点击底部Tab「👤 我的」
2. 在登录卡片中：
   - 输入**姓名**（如：张三）
   - 输入**密码**（初始密码请联系辅导员或系统管理员获取）
3. 点击蓝色「登 录」按钮
4. 登录成功后，页面自动切换为个人资料展示

### 3.2 登录成功后的状态

- 个人资料卡片显示：头像首字母、姓名、学号、专业
- 底部显示详细资料列表：姓名、学号、性别、年级、专业、班级、电话、邮箱
- 课表页自动加载你所在班级的课程数据
- 论坛页右下角出现蓝色「+」发帖按钮

### 3.3 登录失败怎么办

| 错误提示 | 可能原因 | 解决方法 |
|----------|----------|----------|
| "请输入姓名和密码" | 表单为空 | 确认两个字段都已填写 |
| "登录失败" | 姓名或密码错误 | 核对姓名、密码是否正确 |
| "登录失败，请检查网络" | 网络异常 | 检查手机网络连接 |
| "登录已过期，请重新登录" | Token过期 | 重新进入「我的」页登录 |

### 3.4 退出登录

1. 进入「👤 我的」Tab
2. 滑动到底部，点击红色「退出登录」按钮
3. 确认弹窗中选择「确定」
4. 退出后课表恢复为空状态，论坛发帖按钮隐藏

## 四、📅 课表功能 — 操作指引

### 4.1 页面布局说明

```
┌──────────────────────────────┐
│  ‹ 第3周 ›   📝 考试         │  ← 周次选择器 + 考试入口
├──────────────────────────────┤
│      周一  周二  周三 ... 周日 │  ← 星期表头（今天蓝色高亮）
│ 1-2节 高数  英语    ...       │
│ 3-4节 线代  体育    ...       │  ← 课程网格
│ 5-6节                        │
│ 7-8节                        │
│ 9-10节                       │
├──────────────────────────────┤
│    [周视图]   [列表]          │  ← 视图切换
└──────────────────────────────┘
```

### 4.2 切换周次

- 点击 **‹** 箭头：查看上一周课表
- 点击 **›** 箭头：查看下一周课表
- 中间显示当前周次（第1~20周）
- 第1周时左箭头不可点击，第20周时右箭头不可点击

### 4.3 切换视图

底部有两个互斥按钮：

- **周视图**（默认）：二维表格形式，横轴星期、纵轴时间段，直观展示整周课程
- **列表视图**：按天分组，每天列出课程卡片，包含时间、地点、教师信息

### 4.4 查看课程详情

1. 在周视图中点击**蓝色课程格子**
2. 或在列表视图中点击**课程卡片**
3. 弹出课程详情弹窗，显示：
   - 📘 课程名称
   - 👨‍🏫 授课教师
   - 📍 上课地点
   - 📅 上课时间（第X周 周X 第XX节）
   - 🎓 学分
4. 点击「关闭」或弹窗外部区域可关闭

### 4.5 今日高亮

- 当天的星期列表头会以**浅蓝色背景**标识
- 例如：今天是周三，则「周三」列背景为浅蓝色

### 4.6 未登录状态

- 课表网格显示为空（白底灰线）
- 中间覆盖半透明遮罩，显示「📋 暂无课表数据」
- 提示"登录后可查看您的课程安排"
- 点击「去登录」跳转到「我的」Tab

## 五、📝 考试安排 — 操作指引

### 5.1 如何进入

- 方式一：在课表页顶部，点击「📝 考试」按钮
- 方式二：如需添加到Tab栏，请联系开发者配置

### 5.2 页面布局说明

考试安排分为三个区域：

```
┌──────────────────────────────┐
│ ● 今日安排               5月8日│
│ ┃ 高级人工智能导论             │
│ ┃ 📍 理科楼B座402 08:30~10:30 │  ← 今日考试蓝色高亮
│ ┃ 进行中                      │
├──────────────────────────────┤
│ ● 未来安排                    │
│ 2025-05-11                    │
│ ┃ 概率论与数理统计        3天后│
│ 2025-05-13                    │
│ ┃ 大学英语(IV)            5天后│  ← 未来考试正常显示
├──────────────────────────────┤
│ ● 已结束（半透明）             │
│ ┃ 数据库系统概论               │  ← 已结束灰色半透明
└──────────────────────────────┘
```

### 5.3 三个分区的含义

| 分区 | 标识色 | 说明 |
|------|--------|------|
| 今日安排 | 蓝色圆点 + 浅蓝背景 | 当天有考试，卡片高亮，显示"进行中" |
| 未来安排 | 灰色圆点 | 未来的考试，显示倒计时"X天后" |
| 已结束 | 灰色圆点 + 半透明 | 已过去的考试，卡片半透明显示"已结束" |

### 5.4 考试卡片信息

每张卡片包含：
- **课程名称**（加粗大号）
- **📍 考试地点**
- **时间**：开始时间 ~ 结束时间
- **👨‍🏫 授课教师**
- **👥 班级名称**
- **考试类型标签**（如：期末考试、机考）

### 5.5 空状态

如果当前没有考试安排，页面中央显示：
- 📋 图标
- "暂无考试安排"

### 5.6 数据说明

- 已登录：加载你所在班级的真实考试数据
- 未登录或加载失败：自动展示演示数据供预览

## 六、💬 论坛功能 — 操作指引

### 6.1 浏览帖子

1. 点击底部Tab「💬 论坛」
2. 帖子以卡片列表展示，每张卡片包含：
   - **头像**：作者名字首字母（蓝色圆形）
   - **作者名 + 发布时间**
   - **置顶标签**（红色📌 置顶，仅置顶帖显示）
   - **标题**（加粗大字）
   - **正文摘要**（灰色，最多3行）
   - **底部操作栏**：❤️点赞数 | 💬评论数 | 发布时间

### 6.2 搜索帖子

1. 顶部搜索栏输入关键词
2. 点击键盘「确认」键执行搜索
3. 搜索后列表刷新显示匹配结果
4. 点击 ✕ 清除搜索内容，恢复全部帖子

### 6.3 排序切换

搜索栏下方有两个排序按钮：

- **最新**（默认）：按发布时间从新到旧排列
- **最热**：按点赞数从高到低排列
- 点击切换时列表自动刷新

### 6.4 下拉刷新

- 在帖子列表顶部**向下滑动**
- 显示刷新动画，加载最新帖子
- 适用于查看新发布的帖子

### 6.5 上拉加载更多

- 滚动到列表**底部**
- 自动加载下一页帖子（每页10条）
- 直至加载完所有帖子

### 6.6 点赞帖子

- 点击任意帖子卡片底部的 **🤍（空心）** 图标
- 变为 **❤️（红心）**，点赞数+1
- 再次点击取消点赞
- **需要登录**：未登录会弹窗提示「请先登录」
- 点赞状态会持久化保存

### 6.7 查看帖子详情

1. 点击任意帖子卡片
2. 进入帖子详情页，显示：
   - 帖子完整标题和正文
   - **评论区**：每条评论包含头像、作者、时间、内容
   - **底部输入栏**：输入框 + 发送按钮
3. 详情页也可点赞（❤️按钮）

### 6.8 发表评论

1. 进入帖子详情页
2. 在底部输入栏输入评论内容
3. 点击「发送」按钮
4. **需要登录**：未登录时底部显示"登录后才能评论..."
5. 评论成功后自动刷新评论区

### 6.9 发布新帖

1. 在论坛列表页，点击右下角蓝色圆形 **「+」按钮**
2. **需要登录**：未登录时该按钮不显示
3. 进入发帖页面：
   - 输入**帖子标题**（必填，最多100字）
   - 输入**帖子正文**（必填，最多5000字）
4. 点击底部「发布」按钮提交
5. 发布成功后自动返回论坛列表

### 6.10 草稿保存功能

- 在发帖页面点击「取消」按钮
- 如果已输入内容，弹出选择：
  - **保存草稿**：下次打开发帖页自动恢复内容
  - **放弃编辑**：清除内容并返回
- 草稿保存在手机本地

### 6.11 未登录使用说明

- 浏览帖子、搜索、查看详情 **无需登录**
- 点赞、评论、发帖 **需要登录**
- 触发需登录操作时弹出提示，可点击「去登录」跳转

## 八、常见问题与故障排除

### 8.1 课表相关

| 问题 | 解答 |
|------|------|
| 为什么课表是空的？ | 可能原因：①未登录，请先登录；②当前周次没有排课，尝试切换周次；③后端未导入课表数据，联系管理员 |
| 课表加载失败怎么办？ | 检查网络连接，下拉刷新重试，如持续失败联系管理员 |
| 为什么只能看到5个时间段？ | 这是标准大学节次设置（1-2节到9-10节），如课程不在此范围请联系管理员确认 |
| 周六周日有课为什么显示不同？ | 周末课程在表格中正常显示，但空单元格用浅灰背景标识 |

### 8.2 考试相关

| 问题 | 解答 |
|------|------|
| 考试数据不准确？ | 考试数据由教务系统导入，如有疑问请联系辅导员核实 |
| 为什么今天有考试但没显示？ | 确认已登录且classGroupId正确，或联系管理员确认考试数据是否已导入 |
| 演示数据是什么意思？ | 当后端不可用或用户未登录时，展示模拟数据供UI预览 |

### 8.3 论坛相关

| 问题 | 解答 |
|------|------|
| 发帖按钮在哪？ | 必须登录后，右下角才会出现蓝色「+」按钮 |
| 为什么点赞后又变回去了？ | 点赞需要调用后端接口，如网络异常可能未成功持久化 |
| 怎么删除我的帖子？ | 当前版本学生端暂不支持删帖，请通过网页管理端操作 |
| 草稿会丢失吗？ | 草稿保存在手机本地存储，清除微信缓存会丢失 |
| 帖子数量不对？ | 尝试下拉刷新获取最新数据 |

### 8.4 登录相关

| 问题 | 解答 |
|------|------|
| 忘记密码怎么办？ | 请联系辅导员或系统管理员重置密码 |
| 登录提示"登录已过期"？ | Token有效期12小时，过期后需重新登录 |
| 可以多设备同时登录吗？ | 可以，每台设备独立存储登录状态 |
| 为什么打开小程序就自动登录了？ | 上次登录的Token和用户信息保存在手机本地 |

### 8.5 通用问题

| 问题 | 解答 |
|------|------|
| 小程序卡顿/白屏？ | 尝试：①关闭小程序重新打开；②清理微信缓存；③检查手机存储空间 |
| 数据加载慢？ | 检查网络状态（WiFi/4G/5G），这是前后端分离架构，需网络通信 |
| 界面显示异常？ | 确认微信版本为最新，小程序基础库版本≥2.0 |
| 如何反馈问题？ | 在论坛中发帖反馈，或联系系统管理员 |

## 九、客服应答话术模板

### 9.1 欢迎语

> 您好！欢迎使用「AI智能排课」小程序 🎓
> 我可以帮您解决以下问题：
> - 📅 如何查看课表、切换周次

> 进入小程序后，默认就是「📅 课表」页面。如果显示为空，请先点击底部「👤 我的」进行登录。

> - 点击左右箭头切换第1~20周

> 页面分为三部分：今日考试（蓝色高亮）、未来考试（显示倒计时）、已结束考试（半透明）。

> 论坛是同学们的交流社区 💬
> - 浏览帖子和搜索**不需要登录**

### 9.6 问题升级

> 您的问题比较复杂，建议通过以下方式进一步处理：
> 1. 请辅导员或系统管理员在后台查看您的账户状态
> 2. 确认您的学生信息（班级、课程）已在系统中正确录入
> 3. 如需技术支持，请联系系统开发团队

## 十、页面交互细节补充

### 10.1 课程详情弹窗（课表页）

触发方式：点击课表网格中的蓝色课程格子
