# AI智能排课系统 — 学生端微信小程序 V1.0

## 项目概述

基于 AI 的大学智能排课系统配套学生端微信小程序，提供课表查看、考试安排、论坛交流、个人资料管理、AI 智能助手等功能。

- **后端**: Spring Boot 2.7 + JPA + MySQL + JWT + OptaPlanner
- **小程序**: 微信原生框架 (WXML + WXSS + JS)
- **AI 助手**: 阿里云百炼 AppFlow Chat SDK（web-view 嵌入）
- **主色调**: `#4A90D9` 蓝色系

## 模块功能

|------|------|
| 周视图 | 二维表格，横轴周一~周日，竖轴 1-2节~9-10节（附时间段 8:00~9:35） |

| 未登录 | 显示空课表 + 「去登录」遮罩 + 首次弹窗引导 |

**API**: `GET /schedule/listByClassGroupId?classGroupId=xxx`

### 5. 🤖 AI 智能助手

| 功能 | 说明 |
|------|------|
| 入口 | 课表页顶部「🤖 AI助手」按钮 |
| 对话 | 跳转独立页面，通过 web-view 加载阿里云百炼 AppFlow Chat SDK |
| 自动展开 | SDK 加载后自动展开聊天面板，无需手动点击 |
| 返回 | 导航栏自带返回按钮，回到课表页 |

**技术方案**:
- 小程序页面 `pages/ai-assistant/ai-assistant` 使用 `<web-view>` 加载 `chatbot.html`
- `chatbot.html` 由 Spring Boot 静态资源服务（`static/chatbot.html`）
- 加载阿里云百炼 AppflowChatSDK.js，传入 `integrateId` 和 `requestDomain`
- SDK 自动处理认证和对话逻辑
- 页面加载时通过 MutationObserver + 定时扫描自动点击 SDK 按钮展开聊天

**后端接口**:
| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/chatbot/chat` | AI 对话代理（备选，直接调用百炼 API 需登录认证） |
| GET | `/chatbot/anonymous-config` | 获取匿名配置 |
| GET | `/chatbot/session-token` | 获取用户会话 Token（预留） |

**配置文件** (`application.yml`):
```yaml
appflow:
  chat:
    integrate-id: "cit-176c651331b04fa6922a"
    request-domain: "https://1557976991754980.appflow.aliyunnest.com"
    api-key: "sk-1d9d8997da39483598f870d9f57dd942"
    api-path: "/webhook/chatbot/chat/"
```

**后端类**:
- `controller/ChatbotController.java` — AI 助手 REST 接口
- `service/ChatbotService.java` — 百炼 API 调用与模拟回复降级

## 技术要点

- **登录态**: JWT Token 存储于 wx.Storage，请求自动携带 Authorization Header
- **下拉刷新**: 论坛页支持 enablePullDownRefresh
- **点赞持久化**: likedPostIds 存本地，切换排序/刷新后恢复
- **草稿功能**: 发帖取消时可选存草稿，下次打开自动恢复
- **演示兜底**: 后端不可用时自动展示模拟数据，不影响 UI 预览
- **timeSlot 映射**: 后端值 1/3/5/7/9 → 前端 1-2节/3-4节/5-6节/7-8节/9-10节
- **AI 助手**: web-view 嵌入阿里云百炼 AppFlow SDK，后端 ChatbotService 支持百炼 API 调用与模拟回复自动降级
- **基础 URL**: `http://10.17.86.214:8084`（局域网地址，开发环境）
