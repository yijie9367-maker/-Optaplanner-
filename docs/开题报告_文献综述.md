文献综述

1. 教学辅助系统的发展与应用

随着信息技术在高等教育领域的不断渗透，教学辅助系统已成为提升教务管理效率、改善教学体验的重要工具。冯秀萍[1]探讨了基于人工智能的高校计算机专业教学辅助系统的设计思路，指出AI技术在课程推荐、学习行为分析等方面的应用潜力，为智能化教务工具的研发提供了方向性参考。卢道设等[2]研究了移动互联环境下教学辅助系统的应用模式，强调移动端访问和跨平台兼容性对用户体验的重要性。于正旺[3]进一步设计了一种支持智能分析的英语翻译教学辅助系统，将数据分析与教学业务深度融合，验证了智能化功能在特定教学场景中的可行性。这些研究从不同角度揭示了教学辅助系统从信息化向智能化演进的趋势，为排课辅助系统的智能化设计提供了理论基础。

2. 排课问题的求解方法研究

排课问题（Timetabling Problem）是运筹学与教育管理领域长期关注的经典组合优化问题，其核心是在满足教师、教室、班级、时间等多维约束的前提下，为各门课程分配合适的教学时间和地点。由于排课问题的解空间随教学资源数量呈指数级增长，该问题属于NP-hard问题，传统的手工编排和简单规则匹配方法难以保证求解质量。

遗传算法因其良好的全局搜索能力在排课领域得到了最为广泛的研究和应用。刘莉和栗超[6]设计了基于遗传算法的智能排课系统，将排课方案编码为染色体，通过选择、交叉、变异操作在解空间中进行并行搜索，实验证明遗传算法能够在较短时间内生成满足硬约束的可行排课方案。史昊天[10]针对标准遗传算法收敛速度慢、易陷入局部最优的问题，提出了改进遗传算法并在研究生排课与组卷系统中加以应用，通过自适应交叉率和变异率的调整策略提升了算法的收敛性能。周国栋[13]提出了基于改进双种群遗传算法的智能排课方法，将种群划分为两个子种群分别承担全局探索与局部开发的角色，通过种群间的信息交换机制平衡了搜索的广度与深度。朱婷婷[14]将改进遗传算法应用于职业院校自动排课系统，针对职业院校课程类型多样、实训课占比高的特点，设计了专门的编码方案和适应度函数。

国际上，Yan和Deng[11]提出了自适应遗传算法用于教育信息化背景下的计算机课程排课系统，通过动态调整遗传算子参数使算法在求解过程中自适应地平衡探索与开发能力。Haider等[12]进一步探索了遗传算法与深度强化学习相结合的智能排课策略，利用深度强化学习网络指导遗传算法的搜索方向，在复杂约束场景下取得了优于单一算法的求解效果。上述研究充分表明，遗传算法及其改进变体是当前排课求解领域的主流方法，且与强化学习等前沿技术的融合代表了未来的发展方向。

在遗传算法之外，其他智能化方法也被引入排课领域。杜冬霞[7]研究了多Agent通信协商技术在高职院校排课系统中的应用，将课程、教师、教室等资源抽象为独立的Agent实体，通过Agent间的消息传递与协商机制完成排课资源的分配，为排课问题提供了分布式求解的新视角。薛阳[4]基于Java技术开发了沈阳职业技术学院的排课辅助系统，从工程实践角度验证了Java EE技术在教务软件中的适用性。

3. 相关技术基础

在技术实现层面，排课系统的开发涉及数据库设计、编程语言和知识工程等多个技术领域。闫秋艳和孟凡荣[8]编著的《数据库原理与应用（MySQL版）》系统阐述了MySQL数据库的设计原则、SQL编程和性能优化方法，为排课系统的数据持久化层构建提供了技术支撑。王全新[9]编著的《Java语言程序设计》涵盖了Java语言的核心语法和面向对象编程思想，是Spring Boot等Java Web框架的技术基础。吴重光和纳永良[5]的《知识图谱建模与智能推理技术》探讨了知识表示与推理的方法论，课程之间的前置关系、教师专业领域等语义信息可借鉴知识图谱的思想进行结构化建模。刘金亮[15]编著的《Odoo快速入门与实战》介绍了开源ERP系统的开发模式，其中模块化业务组件和权限管理机制的设计思路对排课系统的架构设计具有参考价值。

4. 文献评述与研究定位

综合以上研究现状，可以看到：遗传算法已在排课求解中积累了大量研究成果，从标准遗传算法到改进双种群遗传算法，求解效率与质量持续提升；深度强化学习与遗传算法的融合为排课优化开辟了新的研究方向。在教学辅助系统方面，智能化、移动化、数据分析驱动已成为主流发展趋势，相关研究为本课题的系统设计提供了重要参考。

然而，现有研究仍存在以下可改进之处：第一，多数排课系统的约束配置固化于代码之中，教务人员难以根据学期实际情况灵活调整约束权重和求解参数；第二，排课结果的可视化呈现以纯表格为主，缺乏按教室、教师、班级等多维度切换的直观课表视图及冲突高亮展示；第三，排课系统与考试管理、成绩录入等关联教务功能的集成度不足，难以形成完整的教务管理闭环。本研究将在吸收上述研究成果的基础上，以改进遗传算法为求解核心，采用Spring Boot + Vue 3前后端分离架构，设计并实现一个支持约束权重动态配置、提供多维度课表可视化且集成多种教务辅助功能的一体化智能排课系统。
